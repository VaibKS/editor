Getting Started

First of all, Welcome to enCode!
Using this editor is pretty straight forward, similar to other ones.
This editor project was made so that people who are using Netbooks or Chromebooks could better work on projects while on the go! And also, for anyone with access to internet with a computer.

With this app, you can download the file as is in its file format or the entire Project as a zip file, or in JSON format, so that the project can be loaded in here and you work on it from where you left of!

To see this again, just press Alt + H.

Shortcuts

Key(s) Action
Ctrl + S Saves the current file.
Ctrl + Q Closes the current tab.
Ctrl + G Opens GoTo Dialog to Go To a Line number
Esc Hides the message or dialog box.
Esc + (n) Opens the tab numbered n from the left.
Esc + Q Opens the tab on the left of the active tab.
Esc + W Opens the tab on the right of the active tab.
Alt + H Opens/Closes the Help Center.
Alt + F Toggle Fullscreen View.
